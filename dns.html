Domain Name System

Wenn ein Nutzer über einen Domainnamen auf eine Website zugreifen möchte, werden für die Interaktion zwischen Client (Computer des Nutzers) und Server IP-Adressen benötigt.

Das Domain Name System (DNS) ist ein Verzeichnisdienst im Internet. Im Domain Name System sind für jeden Domainnamen IP-Adressen des entsprechenden Servers hinterlegt. Verschiedene über die Welt verteilte DNS-Server übersetzen die Domainnamen in IP-Adressen.

Beispiel:

www.hansenberg.de

=

178.32.80.254
