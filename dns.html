Domain Name System

Damit die Verbindung zu einem Server aufgebaut werden kann, wird die IP-Adresse des Servers benötigt.

Das Domain Name System übersetzt Domain-Namen in IP-Adressen. Nur deshalb ist es uns also möglich, Websites im Browser zu öffnen ohne die IP-Adresse des entsprechenden Servers auswendig zu wissen.

Beispiel:

www.hansenberg.de

178.32.80.254

Domainnamen

Ein Domainname ist ein weltweit einmaliger Name für einen abgegrenzten Teilbereich des Internets, wie beispielsweise ein Website.

Der Domainname bzw. seine einzelnen Bestandteile geben an, wo die Website im hierarchisch strukturierten Domain Name System zu finden ist.

Top-Level-Domain

Ein Domainname besteht aus Top-Level-Domain, Second-Level-Domain und Third-Level-Domain.

Top-Level-Domain:
Die Top-Level-Domain steht am Ende des Domainnamen. Sie können in generische TLDs, wie .com oder .net, und länderspezifische TLDs, wie .de oder .at, eingeteilt werden.

Second-Level-Domain:
Die Second-Level-Domain steht vor der TLD. Sie ist innerhalb des Namensraums der TLD einmalig.

Third-Level-Domain:
Die Third-Level-Domain steht am Anfang des Domainnamens. Durch die Third-Level-Domain können verschiedene Bereiche einer Domain voneinander getrennt werden. Beispielsweise steht www für Webservices und mail für Mailserver.

Wie funktioniert eine DNS-Abfrage?

  1. Der Nutzer gibt einen Domain-Namen in den Browser ein
  2. Der Resolver schickt eine Anfrage an den DNS-Rootserver.
    Der Resolver ist ein Programm, das die Anfrage an das DNS übernimmt. Er fungiert somit als Schnittstelle zwischen Anwendung und Namensserver.
  3. Der Rootserver gibt dem Resolver an, in welchem Top-Level-Domain sich die Website befindet.
  4. Der Resolver schickt eine Anfrage an einen Server des entsprechenden Top-Level-Domains, um die IP-Adresse des richtigen Namenservers zu ermitteln.
  5. Der Resolver schickt eine Anfrage an den entsprechenden Namenserver, welcher die entsprechende IP-Adresse zurückgibt.
  6. Der Resolver gibt die IP-Adresse an den Browser weiter.
  7. Der Browser schickt eine Anfrage an die erhaltene IP-Adresse.
