Domain Name System

Damit die Verbindung zu einem Server aufgebaut werden kann, wird die IP-Adresse des Servers benötigt.

Das Domain Name System übersetzt Domain-Namen in IP-Adressen.

Beispiel:

www.hansenberg.de

=

178.32.80.254

Domainnamen

Ein Domainname ist ein weltweit einmaliger Name für einen abgegrenzten Teilbereich des Internets, wie beispielsweise ein Website.

Der Domainname bzw. seine einzelnen Bestandteile geben an, wo die Website im hierarchisch strukturierten Domain Name System zu finden ist.

www.hansenberg.de

Third-Level-Domain

Second-Level-Domain

Top-Level-Domain

Ein Domainname besteht aus:

  • Top-Level-Domain
  • Second-Level-Domain
  • Third-Level-Domain

Wie funktioniert eine DNS-Abfrage?

  1. Der Nutzer gibt einen Domain-Namen in den Browser ein

    2. ->

  2. Der Resolver schickt eine Anfrage an den DNS-Rootserver.

    3. ->

  3. Der Rootserver an in welchem Top-Level-Domain sich die Website befindet.

    4. ->

  4. Der Resolver schickt eine Anfrage an einen Server des entsprechenden Top-Level-Domains.

    5. ->

  5. Der Resolver schickt eine Anfrage an den entsprechenden Namenserver.

    6. ->

  6. Der Resolver gibt die IP-Adresse an den Browser weiter.

    7. ->

  7. Der Browser schickt eine Anfrage an die erhaltene IP-Adresse.
