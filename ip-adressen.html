Was ist eine IP-Adresse?

Sollen Daten zwischen verschiedenen Hosts versandt werden benötigen sie eine digitale Adresse: die IP-Adresse.

Jeder Host in einem Netzwerk verfügt über mindestens eine IP-Adresse und kann so identifiziert werden.

Im Gegensatz zu normalen Adressen sind IP-Adressen meist dynamisch. Das heißt, dass sie immer wieder neu zugeteilt werden. Trennt dein Router sich (und somit dein Netzwerk) vom Internet, so werden die IP-Adressen aller Netzwerkteilnehmer neu vergeben. Verbindet sich dein Router nun erneut mit dem Internet, so erhalten alle Netzwerteilnehmer neue IP-Adressen. Lediglich manche Dienste im Internet (z.B. bestimmte Websites) greifen auf immer gleich bleibende statische IP-Adressen zurück.

Der Aufbau der IP-Adresse unterscheidet sich je nach dem, ob es sich um eine IPv4- oder eine IPv6-Adresse handelt.
