Aufbau

Es gibt zwei verschiedene Versionen vom IP -> IPv4- und IPv6-Adressen.

IPv4-Adressen

IPv4-Adressen werden als vier Dezimalzahlen von 0 bis 255 dargestellt.

Somit ist jede der vier Zahlen ein Byte groß.

Beispiel:

192.168.100.120

=

11000000.10101000.01100100.01111000

Die Subnetzmaske

Zu jeder IPv4-Adresse gehört eine Subnetzmaske.

Sie gibt an welcher Teil der IP-Adresse das Netzwerk beschreibt und welcher Teil den Host.

Die Subnetzmaske gleicht in Aufbau und Darstellungsweise einer IP-Adresse.

Wandelt man sie in Binärzahlen um, besteht sie aus einer aufeinanderfolgenden Reihe von 1en und 0en.

An dem Punkt, wo die 1en auf 0en umspringen beginnt der Netzwerkteil der zugehörigen IP-Adresse.

Beispiel:

Subnetzmaske:

255.255.252.0

zugehörige IP-Adresse:

11000000.10101000.01100100.01111000

IPv6-Adressen

IPv6-Adressen bestehen aus 8 Teilen, die eine Größe von 16 Bit haben. -> Insgesamt 128 Bit

Je 4 Bit werden als einstellige Hexadezimalzahl dargestellt. -> vierstellige Hexadezimalzahl

Abkürzen

Zusammenhängende Blöcke, die ausschließlich aus 0en bestehen können einmal durch zwei Doppelpunkte bagekürzt werden.

Andere Blöcke, die aus 0en bestehen, können zu einer einzigen 0 abgekürzt werden.

Beispiel

2a02:0000:973f:e78c:0000:0000:0000:e061

Der Präfix

Auch eine IPv6-Adresse kann in einen Netzwerk = Präfix und einen Hostteil unterteilt werden.

Statt der Subnetzmaske gibt es die Präfixlänge: Sie gibt an wie lange der Präfix ist.

Beispiel:

2a02:8108:973f:e78c:7549:1759:ae97:e061/64
